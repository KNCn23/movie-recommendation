# 🎬 Movie Recommendation System (Film Öneri Sistemi)

[![Python](https://img.shields.io/badge/Python-3.x-blue.svg)](https://www.python.org/)
[![Pandas](https://img.shields.io/badge/Data-Pandas-orange.svg)](https://pandas.pydata.org/)
[![Scikit-Learn](https://img.shields.io/badge/ML-Scikit--Learn-green.svg)](https://scikit-learn.org/)

[TR] Netflix veri seti (netflix_titles.csv) üzerinden içerik tabanlı (Content-Based) film önerileri sunan bir Python projesi. TF-IDF ve Cosine Similarity algoritmalarını kullanır.

[EN] A Python project providing content-based movie recommendations using the Netflix dataset (netflix_titles.csv). It utilizes TF-IDF and Cosine Similarity algorithms.

---

## ✨ Features / Özellikler

- **[TR] İçerik Tabanlı Filtreleme:** Film açıklamalarından benzerlik analizi yapar.
- **[EN] Content-Based Filtering:** Analyzes similarities based on movie descriptions.
- **[TR] Veri Analizi:** Pandas kütüphanesi ile veri temizleme ve işleme.
- **[EN] Data Analysis:** Data cleaning and processing using Pandas.

---

## 🚀 Setup & Run / Kurulum ve Çalıştırma

### [TR] Adımlar
1. Gerekli kütüphaneleri yükleyin: `pip install pandas scikit-learn`
2. `netflix_titles.csv` dosyasının dizinde olduğundan emin olun.
3. Çalıştırın: `python movie_recommender.py`

### [EN] Steps
1. Install dependencies: `pip install pandas scikit-learn`
2. Ensure `netflix_titles.csv` is in the directory.
3. Run: `python movie_recommender.py`
